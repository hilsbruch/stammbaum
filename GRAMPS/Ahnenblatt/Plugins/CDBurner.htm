Burning a CD with Ahnenblatt ...

By starting "Tools / Burn CD" the following files will be saved on a CD:
  • HTML-Website of presently opened Ahnenblatt-file (pictures included if exist)
  • the Ahnenblatt-file itself
  • the Ahnenblatt-file converted to GEDCOM format (for using with other genealogical software)
  • icon file
  • autostart file for CD
When finished, the output folder is opened.

At the end you can burn the CD with a burning software of your choice.
