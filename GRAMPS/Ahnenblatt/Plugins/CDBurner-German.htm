Brennen einer CD mit Ahnenblatt ...

Durch Aufruf von "Tools / CD brennen" werden folgende Dateien in ein Verzeichnis gespeichert:
  • HTML-Website der aktuell geöffneten Ahnendatei (falls vorhanden inkl. Bilder)
  • die Ahnendatei selbst
  • die Ahnendatei im Gedcom-Format (zur Verwendung in anderen Programmen)
  • Icon-Datei
  • Autostart-Datei für CD
Am Ende wird das Zielverzeichnis geöffnet.

Der Inhalt des Verzeichnisses kann jetzt manuell mit einem eigenen CD-Brennprogramm auf CD gebrannt werden.
