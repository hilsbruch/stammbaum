Gravure de CD avec l'application Ahnenblatt ...

Le fait d'appeler la focntion "Outils / Gravure CD" les fichiers suivants sont sauvegardés dans un répertoire:
  • HTML - Page Web du fichier des ancêtres en cours ( y compris les images si celles ci sont disponibles)
  • le fichier des ancêtres
  • le fichier des anc^tres au format GEDCOM (pour de futures utilisations)
  • Fichier des icônes
  • Fichier de démarrage pour CD (Autorun)
Pour terminer le répertoire de destination sera ouvert.

Le contenu du répertoire peut alors être gravé manuellement à l'aide d'un programme standard de gravure.
