Gravure de CD avec l'application Ahnenblatt ...

L'appel de la fonction "Outils/ Gravure CD (Windows) " les fichiers suivants sont copiés dans un répertoire Windows à des fins de gravure (ne fonctionne qu'avec Windows XP):
  • HTML - Page Web du fichier des ancêtres en cours ( y compris les images si celles ci sont disponibles)
  • le fichier des ancêtres
  • le fichier des anc^tres au format GEDCOM (pour de futures utilisations)
  • Fichier des icônes
  • Fichier de démarrage pour CD (Autorun)
Pour terminer le répertoire de destination sera ouvert.

En règle générale il suffira de mettre un CD vierge dans le graveur et un assistant Windows vous guidera pour la gravure.

Aprés gravure avec succés Windows effacera automatiquement les fichiers.
